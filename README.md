# JiraService
С помощью данного сервиса можно получать данные из Jira API и строить графики

## Использование приложения

### time-spent 
Построение гистограммы, отражающей время, которое задача провела
в открытом состоянии (от момента создания до момента закрытия). По оси
абсцисс – расположить время. По оси ординат – расположить суммарное
количество задач, которое было в открытом виде соответствующее время. В
расчет брать только закрытые задачи по проекту

#### Пример запуск
    java -jar JiraService.jar time-spent issues.apache.org KAFKA TimeSpent
time-spent - имя утилиты.
issues.apache.org - домен Jira API.
KAFKA - имя проекта.
TimeSpent - заголовок графика.

### quantity-issue
Строит график, показывающий количество заведенных и закрытых
задач в день. График должен отражать не только информацию о количестве
задач в день, но и накопительный итог по задачам. По оси абсцисс –
расположить календарные дни. По оси ординат – расположить суммарное
количество задач для этого дня

#### Пример работы
    java -jar JiraService.jar quantity-issue issues.apache.org KAFKA issue 2023-11-01 2023-11-30
quantity-issue - имя утилиты.
issues.apache.org - домен Jira API.
KAFKA - имя проекта.
issue - заголовок графика.
2023-11-01 2023-11-30 - интервал дат.

### top-users
Строит график, выражающий общее количество задач для
пользователей, в которых он указан как исполнитель и репортер. По оси
абсцисс – расположить количество задач. По оси ординат – расположить имя
пользователя.

#### Пример работы
    java -jar JiraService.jar top-users issues.apache.org KAFKA topUsers 30
top-users - имя утилиты.
issues.apache.org - домен Jira API.
KAFKA - имя проекта.
topUsers - заголовок графика.
30 - количество пользователей

### user-time-spent
Построение гистограмму, отражающую время, которое затратил
пользователь на ее выполнение на основе залогированного времени. По оси
абсцисс – расположить время. По оси ординат – расположить суммарное
количество задач, которое соответствует этому времени.

#### Пример работы
    java -jar JiraService.jar user-time-spent issues.apache.org KAFKA UserTimeSpent lihaosky
user-time-spent - имя утилиты.
issues.apache.org - домен Jira API.
KAFKA - имя проекта.
UserTimeSpent - заголовок графика.
lihaosky - имя пользователя

### issue-priorities
Строить график, выражающий количество задач по степени
серьезности.

#### Пример работы
    java -jar JiraService.jar issue-priorities issues.apache.org KAFKA priorities
issue-priorities - имя утилиты.
issues.apache.org - домен Jira API.
KAFKA - имя проекта.
priorities - заголовок графика.
